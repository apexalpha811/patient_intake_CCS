# Culver City Surgical — Patient Intake Form

A bilingual (English/Spanish) fillable patient intake form for Culver City Surgical Specialists, designed to match the official brand design system.

## Deployment

This is a static HTML site — deploy by pointing Vercel at this repo. No build step required.

## Files

- `index.html` — the patient intake form
- `Design System/fonts/` — Fraunces + Inter web fonts
- `Design System/assets/logo/` — Culver City Surgical logos
